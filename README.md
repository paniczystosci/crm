# Pani Czystości CRM

Система управления клининговой компанией. Администраторы могут управлять заказами, клинерами и финансами. Клинеры получают задания, общаются в чате и сдают кассу.

## 🚀 Основные возможности

### Для администратора
- **Управление заказами** — просмотр, создание, редактирование, изменение статуса.
- **Управление клинерами** — добавление сотрудников, настройка ставки (15%, 25%, 50% или ручная).
- **Выплаты** — отметка выплат клинерам, учёт наличных и безналичных платежей.
- **Статистика** — графики доходов, распределение заказов по статусам, последние выплаты.
- **Чат с клинером** — общение в реальном времени по каждому заказу.

### Для клинера
- **Мои заказы** — список заданий с фильтрацией по статусу.
- **Новый заказ** — создание заказа с проверкой доступных слотов.
- **Касса** — учёт наличных и безналичных средств, сдача кассы.
- **Чат с администратором** — оперативное общение по каждому заказу.

### Общие функции
- **Уведомления в реальном времени** — звук, бейдж в меню, обновление заголовка вкладки.
- **Тёмная/светлая тема** — сохраняется в localStorage.
- **Адаптивный дизайн** — удобная работа с телефона, планшета, ПК.

## 📦 Технологии

- **Next.js 16** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS** (с поддержкой тёмной темы)
- **Supabase** (аутентификация, база данных, realtime, storage)
- **Lucide React** (иконки)
- **Recharts** (графики статистики)

## 📋 Требования

- Node.js 18+ и npm
- Аккаунт Supabase (бесплатный тариф подойдёт)
